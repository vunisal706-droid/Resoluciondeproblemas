✅ SISTEMA ANTI-REPETICIÓN
- Cada plantilla tiene múltiples rangos de números
- Los números se generan aleatoriamente cada vez
- Variedad garantizada en cada sesión

✅ FASES DE RESOLUCIÓN
1. COMPRENDER → Área de texto para anotar datos
2. PLANIFICAR → Pizarra digital (canvas) para estrategia
3. EJECUTAR → Pizarra digital (canvas) para cálculos
4. VERIFICAR → Pizarra digital (canvas) para comprobación
5. SOLUCIÓN FINAL → Área de texto para respuesta definitiva

✅ HERRAMIENTAS DE DIBUJO
- Lápiz con 6 colores diferentes
- Borrador
- Limpiar canvas completo
- 4 grosores de trazo (2px, 3px, 5px, 8px)
- Soporte táctil para tablets

✅ DISTRIBUCIÓN POR TIPO
Cada mes incluye 2-3 tipos diferentes de problemas:
- Problemas de razonamiento
- Problemas de vida cotidiana
- Retos lógicos

✅ NIVELES DE DIFICULTAD PROGRESIVOS
- 1º Primaria: Números hasta 20
- 2º Primaria: Números hasta 50, multiplicación básica
- 3º Primaria: Números hasta 100, fracciones simples

- 5º Primaria: Operaciones complejas, porcentajes
